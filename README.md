## Installation

1. Install theme from the [Marketplace](https://marketplace.visualstudio.com/items?itemName=StanislavKorchevskyi.hybrid-dark-theme)
2. Go to `File > Preferences > Color Theme`
3. Select `Hybrid Dark Theme`
